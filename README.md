# viloagents

A modular AI chat backend based on FastAPI and LangChain.

## Development Setup

1. Install dependencies using Poetry:
   ```bash
   poetry install
   ```
2. Run the API server:
   ```bash
   poetry run python -m ai_chat_backend.main
   ```
3. Or start the CLI:
   ```bash
   poetry run python -m ai_chat_backend.api.cli
   ```
